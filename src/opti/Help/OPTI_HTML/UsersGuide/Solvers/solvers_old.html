Optimization Solvers

The core function of OPTI Toolbox is to provide a high level interface to the large number of solvers available for optimization. While the focus of the toolbox is on the open-source community solvers, several commercial solvers have been interface as well where academic licenses are available.

The following two pages contain detailed information about the solvers supplied and interface to the OPTI Toolbox. At the bottom of the page is a table containing a solver versus problem matrix.
