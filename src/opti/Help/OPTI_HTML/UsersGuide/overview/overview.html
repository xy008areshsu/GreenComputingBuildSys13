Optimization Problems Overview

Several types of optimization problems can be formulated and solved using the solvers supplied with OPTI Toolbox. Each of the links below will take you to the corresponding page on the Wiki.

Linear Objective

  • System of Linear Equations (SLE)

  • Linear Program (LP)

  • Mixed Integer Linear Program (MILP)

  • Semidefinite Program (SDP)

Quadratic Objective

  • Quadratic Program (QP)

  • Quadratically Constrained Quadratic Program (QCQP)

  • Mixed Integer Quadratic Program (MIQP)

  • Mixed Integer Quadratically Constrained Quadratic Program (MIQCQP)

Nonlinear Objective

  • System of Nonlinear Equations (SNLE)

  • Nonlinear Least Squares (NLS)

  • Nonlinear Program (NLP)

  • Global Nonlinear Program (GNLP)

  • Mixed Integer Nonlinear Program (MINLP)
