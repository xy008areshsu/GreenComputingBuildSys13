Bug Reporting

In order for me to best help you if you have a problem, or find a bug with OPTI, please complete the following steps:

1) Run opti_debug.m and attach the generated opti_debug.txt to a new topic on the OPTI forum. Don't worry no personally identifiable information is contained in the the text file!
2) Provide a code example of how to generate the problem as a m file or copy the contents to the post.
3) If your optimization problem is of a reasonable size, include the problem as a .mat file if possible. If I can run your problem I have a much better chance of fixing it!
4) Attach any relevant screenshots of errors.
