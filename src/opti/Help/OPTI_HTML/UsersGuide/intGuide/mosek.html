MOSEK Installation Guide

MOSEK is a powerful linear and quadratic solver with facilities for discrete variables, quadratic constraints, and separable nonlinear functions. This guide was written for MOSEK 6.0.0.135.

Obtaining MOSEK

1) Visit the MOSEK Academic License Request page and complete your details. The license will be emailed to you.
2) Visit the MOSEK Download page and download the required installer.

Installing MOSEK

3) Launch the MOSEK installer package from your download location and complete the installation. As with other solvers, I typically install to the default location. MOSEK requires a restart after installation.
4) Generate a 'mosek.lic' file with the text copied from your MOSEK license email. I used notepad++ for this task. Copy your mosek.lic file into the 'licenses' directory of your MOSEK installation. On my PC this was:

C:\Program Files\Mosek\6\licenses

Interfacing to OPTI Toolbox

5) MOSEK is supplied with a MATLAB Toolbox (2009b) however if you use it you will break the MATLAB Optimization Toolbox. Therefore do not add the MOSEK MATLAB Toolbox to the MATLAB path! If you have already done so, remove it! Instead we only require one file from this, and the rest I have re-written for use in OPTI. Copy 'mosekopt.mexw64' (or .mexw32) from the MOSEK toolbox, e.g.:

C:\Program Files\Mosek\6\toolbox\r2009b

And paste into the MOSEK folder of your OPTI Installation, e.g.:

OptiToolbox_edu_v1.33\Solvers\mosek

Type the following to test it is working:

>> mosekopt
