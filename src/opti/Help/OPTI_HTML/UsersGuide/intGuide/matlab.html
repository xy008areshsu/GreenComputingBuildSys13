MATLAB Optimization Toolbox Installation Guide

The MATLAB Optimization Toolbox will be automatically detected by OPTI if it is installed with MATLAB. The user should not need to do install anything or add any paths.

NOTE: If you have installed e.g. the MOSEK MATLAB Toolbox or TOMLAB several Optimization Toolbox functions may be overloaded (such as linprog, quadprog). If this is the case OPTI cannot determine the location of the true MATLAB Optimization Toolbox and will call the incorrect method. It is suggested not to overload original MATLAB functions as this can cause compatibility issues between MATLAB versions.
